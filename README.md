# autosurf

## Usage

### Include on page to test

```html
<script src="https://cdn.jsdelivr.net/npm/@dscribers/autosurf/dist/autosurf.min.js"></script>

<script>
    AutoSurf.parse(feature).start()
</script>
```

### Use nodejs

```js
import AutoSurf from '@dscribers/autosurf'

AutoSurf.parse(feature).start()

```
